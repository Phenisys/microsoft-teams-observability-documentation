L'intégration Splunk achemine la télémétrie MS Teams vers Splunk via **HTTP Event Collector (HEC)**, avec une application Splunk dédiée fournissant des tableaux de bord opérationnels.

## Architecture

```
ms-teams-agent (Collector)
        │
        │  HTTPS / HEC
        ▼
  Splunk HEC endpoint (index ms_teams)
        │
        ▼
  Application MS Teams Observability Splunk (tableaux de bord)
```
