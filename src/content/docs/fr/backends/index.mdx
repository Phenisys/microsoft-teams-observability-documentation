Le collector exporte la télémétrie Teams vers un ou plusieurs **backends**. Chaque intégration couvre à la fois la configuration du côté récepteur (configuration de la plateforme) et la configuration de l'output du collector.

## Backends supportés

| Backend | Type | Application incluse |
|---|---|---|
| **Dynatrace** | Plateforme d'observabilité SaaS | Oui — stockage Grail + application dédiée avec dashboards |
| **Splunk** | Plateforme SIEM / analytique | Oui — application Splunk avec ingestion HEC |
| **OTel / OTLP** | Standard ouvert | Non — export vers tout endpoint compatible |

## Ajouter un backend

Le collector supporte plusieurs outputs simultanés. Vous pouvez activer plusieurs backends dans votre configuration en ajoutant plusieurs entrées dans la section `output`. Voir <a href={`${import.meta.env.BASE_URL}fr/collector/configuration/`}>Configuration du Collector</a> pour les détails.
