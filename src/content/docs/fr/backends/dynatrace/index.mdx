L'intégration Dynatrace est le backend le plus complet pour MS Teams Observability. Elle couvre :

- **Stockage** : bucket Grail pour les données de logs Teams.
- **Traitement** : OpenPipeline pour l'extraction de métriques, le routage et l'enrichissement.
- **Application** : application dédiée avec tableaux de bord opérationnels pour la qualité des appels, les performances des sites, l'activité des utilisateurs et la corrélation avec les incidents Microsoft.

## Architecture

```
ms-teams-agent (Collector)
        │
        │  HTTPS / Dynatrace Log Ingest API
        ▼
  Dynatrace Grail (bucket ms_teams)
        │
        │  OpenPipeline (routage + extraction de métriques)
        ▼
  Application MS Teams Observability (tableaux de bord)
```
