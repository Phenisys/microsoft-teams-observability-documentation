The collector exports Teams telemetry to one or more **backends**. Each backend integration covers both the receiving side (platform configuration) and the collector output configuration.

## Supported Backends

| Backend | Type | Application included |
|---|---|---|
| **Dynatrace** | SaaS observability platform | Yes — Grail storage + dedicated dashboards app |
| **Splunk** | SIEM / analytics platform | Yes — Splunk app with HEC ingestion |

## Adding a New Backend

The collector supports multiple simultaneous outputs. You can enable more than one backend in your configuration by adding multiple entries under the `output` section. See <a href={`${import.meta.env.BASE_URL}collector/v1/configuration/`}>Configuration</a> for details.
