The Splunk integration delivers MS Teams telemetry to Splunk via **HTTP Event Collector (HEC)**, with a dedicated Splunk app providing operational dashboards.

## Architecture

```
ms-teams-agent (Collector)
        │
        │  HTTPS / HEC
        ▼
  Splunk HEC endpoint (ms_teams index)
        │
        ▼
  MS Teams Observability Splunk App (dashboards)
```
