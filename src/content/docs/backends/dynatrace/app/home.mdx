## What This Page Shows

The Home page is the global summary view of your tenant over the last 7 days.

It displays:
- overall call volume,
- proportion of healthy calls,
- number of unique callers,
- quality distribution,
- platform distribution.

There are no manual filters on this page. Data is loaded automatically for the fixed 7-day window.

## KPI Cards

### Call Count

- Total number of calls in the last 7 days.
- Includes sparkline trend.
- Useful to detect sudden drops or spikes in usage.

### Healthy Calls

- Percentage of Good calls.
- Color-coded value (green or red).
- This is the fastest indicator to detect a quality degradation.

### Caller Count

- Total unique callers in the last 7 days.
- Includes sparkline trend.
- Helps distinguish whether quality changes come from traffic volume or user impact.

## Call Health Chart

This chart shows how calls are split by health status in the same 7-day window.

| Bar | Color | Meaning |
|-----|-------|---------|
| Good | Green | Calls meeting thresholds |
| Poor | Red | Calls with degradation |

## Platforms Chart

This chart shows which client platforms are currently dominant in your call traffic.

| Slice | Platform |
|-------|----------|
| Windows | Windows desktop client |
| macOS | macOS desktop client |
| iOS | iPhone/iPad app |
| Android | Android app |
| Web | Browser client |
| Unknown | Platform not determined |

## States

All cards/charts share the same loading and empty-data behavior:

| State | Behavior |
|-------|----------|
| Loading | Spinners in cards/charts |
| No data | `No data available` |
| Data loaded | Metrics and charts visible |
